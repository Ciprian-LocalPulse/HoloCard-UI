# Code of Conduct

HoloCard UI is a public-facing premium UI product repository with frontend, portfolio, commercial, and brand-presentation context. All public interaction should remain professional, respectful, security-conscious, and legally calm.

## Expected Conduct

Participants are expected to:

- communicate respectfully and in good faith
- keep discussions focused on public documentation, governance, repository quality, and lawful frontend product use cases
- avoid publishing sensitive commercial, customer, payment, licensing, or private asset details in public channels
- respect the Public Edition and Private Commercial Edition boundary
- disclose security concerns through SECURITY.md rather than public issues
- use precise, professional language suitable for buyers, designers, frontend engineers, procurement teams, and enterprise evaluators

## Unacceptable Conduct

The following behavior is not acceptable:

- harassment, personal attacks, discriminatory language, or intimidation
- requests for credential misuse, payment abuse, deceptive resale, buyer-data exposure, or bypass guidance
- publication of secrets, API keys, tokens, buyer records, transaction details, license keys, private assets, or private implementation details
- attempts to extract private commercial packages, unreleased design variants, source design files, audio internals, premium assets, buyer-only deliverables, or commercial implementation details
- spam, impersonation, misleading claims, or inflammatory language
- public vulnerability disclosure before coordination with the maintainer

## Security-Sensitive Communication

Do not post sensitive reports publicly. Use the process in SECURITY.md for vulnerability reports or repository safety concerns.

If a public issue contains sensitive information, the maintainer may edit, hide, lock, or remove it to protect users, buyers, customers, and the integrity of the product.

## Enforcement

The maintainer may take appropriate action to keep the repository safe and professional, including closing issues, declining pull requests, removing unsafe content, or restricting participation.

Enforcement decisions are made at the maintainer's discretion and are intended to preserve the lawful, enterprise-safe, premium-product scope of the repository.

## Scope

This Code of Conduct applies to public repository interactions, including issues, pull requests, discussions, comments, and any public communication connected to HoloCard UI.
