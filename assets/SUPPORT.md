# Support

## Public Support Scope

Public support for HoloCard UI is limited to:

- public repository navigation
- public documentation questions
- public static preview issues
- broken public links
- clarification of the Public Edition boundary
- responsible disclosure routing

Public support does not include private implementation support, private customization work, private package delivery, customer licensing records, resale authorization, white-label authorization, analytics setup, tracking setup, or commercial implementation detail.

## Private Enterprise Discussions

Private enterprise access, licensing, acquisition, design-partner, customization, or implementation discussions must be handled outside public GitHub channels.

## Security Support

For security concerns, do not open a public issue.

Follow [SECURITY.md](SECURITY.md).

## Response Expectations

This repository does not provide guaranteed public support response times.

Any paid support, private enterprise assistance, customization support, private package support, implementation support, or transfer support must be defined in a separate written agreement.

## Public Communication Rules

Do not post any of the following in public issues, discussions, or pull requests:

- secrets, tokens, keys, or credentials
- customer or buyer data
- private licensing records
- private package assets
- private support material
- prompts or AI orchestration details
- analytics identifiers
- tracking configurations
- third-party confidential information
