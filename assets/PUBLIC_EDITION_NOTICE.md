# Public Edition Notice

## Purpose

This repository is the Public Edition of HoloCard UI.

It exists to provide a procurement-safe, public-facing view of the static UI component, product concept, brand, governance posture, and evaluation surface.

## Public Edition Includes

The Public Edition may include:

- static preview source
- public landing page materials
- public visual assets
- non-sensitive product overview
- ownership and licensing notices
- support and security policies
- contribution rules
- public funding configuration

## Public Edition Excludes

The Public Edition does not include:

- private package assets
- private customer or buyer data
- private licensing records
- private support material
- prompts or AI orchestration details
- API keys, secrets, credentials, or deployment variables
- analytics identifiers or tracking configurations
- internal business strategy
- commercial implementation details
- private distribution workflows
- restricted private enterprise materials

## Private Enterprise Edition

The Private Enterprise Edition or commercial package is separate from this public repository.

Private package access, customization support, commercial usage rights, white-label rights, sublicensing rights, exclusivity, transfer terms, and support obligations are available only through explicit written agreement or authorized purchase terms.

The public repository should not be treated as the complete commercial package, a complete source package, a security system, an identity verification system, a tracking system, an anti-theft system, or a production deployment package.

## Public Discussion Boundary

Public issues and pull requests must remain limited to the public documentation and public preview surface.

Do not request or disclose private package assets, customer data, buyer data, licensing records, prompts, credentials, analytics identifiers, private distribution workflows, internal strategy, or restricted implementation details in public GitHub channels.

## Procurement-Safe Interpretation

This repository may support initial review, but it is not:

- a security audit
- a compliance certification
- an identity verification system
- a payment system
- a tracking system
- an anti-theft system
- a production readiness attestation
- a transfer agreement
- a complete commercial package

Formal diligence, legal review, implementation review, privacy review, or enterprise access review must occur privately under appropriate written terms.
