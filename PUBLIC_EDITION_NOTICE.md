# Public Edition Notice

HoloCard UI is published here as a Public Edition for premium frontend product evaluation, public documentation, portfolio review, UI component discussion, and commercial discovery.

## Purpose of the Public Edition

The Public Edition is intended to communicate:

- premium UI product direction
- public frontend component quality
- portfolio and brand presentation value
- public governance posture
- procurement-safe and buyer-friendly evaluation material
- commercial appeal without exposing private commercial materials

## What the Public Edition May Include

The public repository may include:

- README and public product positioning
- static landing page or public component preview
- public source materials intentionally published in this repository
- non-operational examples
- governance documents
- safe support and security reporting instructions
- high-level commercial evaluation language

## What the Public Edition Does Not Include

The Public Edition does not include and should not be used to infer:

- private commercial package files or buyer-only deliverables
- unreleased design variants, visual systems, animation files, or premium assets
- private audio design notes, audio implementation variants, or buyer-specific behavior changes
- private payment, licensing, fulfillment, analytics, or customer-support implementation details
- customer data, buyer information, transaction records, license keys, or commercial account details
- API keys, tokens, service-account files, credentials, secrets, or environment variables
- private prompts, creative briefs, launch materials, pricing strategy, or commercial negotiation details

## Private Commercial Edition

The Private Commercial Edition, if made available, is separate from this public repository and may only be accessed through an authorized commercial purchase or written agreement.

Private Commercial Edition discussions should remain private and should not be opened as public GitHub issues when they involve pricing, buyer requirements, paid package delivery, private assets, customer information, licensing details, or confidential evaluation criteria.

## Responsible Use

HoloCard UI is positioned for lawful UI evaluation, portfolio presentation, frontend component review, and commercial product discussion. Public materials must not be used to support credential misuse, payment abuse, buyer-data exposure, deceptive resale, or activity outside a lawful and authorized context.

## Public Communication Standard

Issues, pull requests, discussions, and references to this repository should remain calm, factual, enterprise-safe, premium-product appropriate, and security-conscious. Do not publish sensitive findings or private implementation assumptions in public channels.
